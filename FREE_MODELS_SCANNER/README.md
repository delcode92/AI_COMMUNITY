# Free Models Scanner

**Context:** Discovering and evaluating open-source AI models for local deployment.

## Purpose

This folder explores free, open-weight models that can power AI agents without relying on paid APIs.

## Key Considerations

### Model Categories
- **General-purpose LLMs**: Conversational AI, reasoning
- **Small language models (SLMs)**: Faster inference, lower cost
- **Specialized models**: Code generation, RAG-specific, tool-use optimized
- **Embedding models**: For retrieval and semantic search

### Evaluation Criteria
- Performance benchmarks (MMLU, HumanEval, etc.)
- Context window size
- Inference speed and memory requirements
- License terms for commercial use
- Quantization support (GGUF, FP4, etc.)

## Related Topics

- Model quantization and optimization
- Local inference servers (Ollama, vLLM, LM Studio)
- Model comparison and benchmarking
- Cost-benefit analysis vs. API models
