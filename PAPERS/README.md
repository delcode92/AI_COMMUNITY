# AI Research Papers

**Context:** Key research papers for understanding AI agent technology.

## Reading List

### Agent Architecture
- **Generative Agents** (Stanford, 2023) - Agent memory and behavior
- **AutoGen** (Microsoft) - Multi-agent conversations
- **LangChain** documentation - Agent patterns

### Tool Use & Planning
- **Toolformer** (Facebook) - LLMs learning to use tools
- **ReAct** (Yale) - Reasoning + Acting framework
- **Plan-and-Solve** (Google) - Hierarchical planning

### Retrieval & RAG
- **RAG original paper** - Retrieval-Augmented Generation
- **Self-RAG** - Self-refinement for RAG
- **HyDE** - Hypothetical document embeddings

### Evaluation
- **AgentBench** - Evaluating agents on real tasks
- **SWE-bench** - Evaluating agent coding capabilities

## How to Use

1. Read abstract and introduction
2. Skim methodology for implementation ideas
3. Review experiments for performance expectations
4. Note code implementations if available
