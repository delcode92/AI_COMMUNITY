# Algorithm Fundamentals for AI Agents

**Context:** Algorithmic principles underlying agent operations.

## Core Algorithms

### Search Algorithms
- **Breadth-First/Depth-First**: For decision tree exploration
- **A* (A-Star)**: Optimal path finding
- **Monte Carlo Tree Search**: Game AI and planning

### Optimization
- **Gradient Descent**: Model training
- **Simulated Annealing**: Finding global optima
- **Genetic Algorithms**: Evolutionary optimization

### Information Retrieval
- **TF-IDF**: Term frequency scoring
- **BM25**: Improved keyword search
- **Vector Similarity**: Cosine, Euclidean distance

### Caching Strategies
- **LRU (Least Recently Used)**: Cache eviction
- **TTL (Time To Live)**: Expiration-based
- **Write-through/Write-back**: Consistency models

## Agent-Specific Algorithms

### Planning
- **HTN (Hierarchical Task Network)**: Decomposing complex tasks
- **STRIPS**: classical planning representation
- **Graph-based planning**: State-space exploration

### Decision Making
- **Markov Decision Processes**: Sequential decision making
- **Reinforcement Learning**: Learning from rewards
- **Multi-armed Bandit**: Exploration vs exploitation

## Complexity Considerations

- Time complexity: How scaling affects performance
- Space complexity: Memory requirements
- Trade-offs: Accuracy vs speed, consistency vs availability
