# CLI Agent Skeleton

**Context:** Template for building command-line AI agents.

## Purpose

This skeleton provides a foundation for creating interactive CLI-based AI agents.

## Features to Implement

- Command parsing and input handling
- Tool registration mechanism
- Streaming responses
- Multi-turn conversation state
- Escape and interrupt handling

## Example Structure

```bash
./agent --tool=greet --tool=calculate
./agent --chat    # Interactive mode
./agent --help    # List available tools
```

## Use Cases

- Quick testing of agent behaviors
- Debugging tool calls
- Learning agent patterns in terminal
